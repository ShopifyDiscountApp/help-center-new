Quick solutions to the most common issues with Optionify.

## Options Not Appearing on Products

### Check 1: Option is Activated
- Open the option in dashboard
- Verify toggle is ON
- Click Save

### Check 2: Assignment is Correct
- Verify product has correct tag
- Or is in correct collection
- Or is individually selected

### Check 3: Clear Cache
1. Go to Settings
2. Click "Clear Cache"
3. Refresh product page


## Option Data Not in Orders

### Solution
- Ensure customer entered data BEFORE adding to cart
- Check order custom attributes section
- Verify option was active when order placed

## Theme Compatibility Issues

If options don't display correctly:

1. Disable app
2. Re-enable app
3. Click "Reinstall Theme Integration"
4. Test on product page


## Character Counter Not Working

- Enable JavaScript
- Update browser
- Check theme compatibility
- Contact support if persists

## Upload Errors

For image upload options:

- Check file size (must be under limit)
- Verify file type is allowed
- Ensure stable internet connection

## Getting Help

1. Search this help center
2. Check documentation
3. Contact support via app
4. Email: support@optionify.com


## Common Questions

**Q: Can I use Optionify with variants?**
A: Yes! They work together perfectly.

**Q: Is there a limit on options?**
A: No, create unlimited options.

**Q: Does it work on mobile?**
A: Yes, fully responsive.

**Q: Can I export option data?**
A: Yes, included in order exports.
