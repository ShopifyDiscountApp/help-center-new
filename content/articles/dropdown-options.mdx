Dropdown options create a select menu for choosing from many options without cluttering the page.

## Use Cases

- Country/state selection
- Long lists of colors
- Product variants
- Time slots


## Configuration

Add options to dropdown:

```
- Select a Color -
Red
Blue
Green
Yellow
Custom Color (+$5)
```
