Switch options provide a simple toggle for binary (yes/no) choices. Clean, modern UI for on/off decisions.

## Use Cases

- Add gift wrapping (Yes/No)
- Include batteries (Yes/No)
- Subscribe to newsletter (Yes/No)
- Express shipping (Yes/No)


## Configuration

```
Label: "Add Gift Wrapping"
Price: $5.00 (when enabled)
Default: Off
```
