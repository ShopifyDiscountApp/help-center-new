Radio button options allow customers to select exactly one choice from multiple options. Perfect for size selection, style choices, and exclusive preferences.

## Use Cases

- Size selection (Small, Medium, Large)
- Style variants (Classic, Modern, Vintage)
- Shipping speed (Standard, Express, Overnight)
- Gift wrapping options (Yes, No)


## Creating Radio Options

1. Select **Radio** option type
2. Add your choices (e.g., Small, Medium, Large)
3. Set default selection (optional)
4. Add pricing to individual choices if needed
