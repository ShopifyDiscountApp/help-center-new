Color dropdown options let customers choose colors from presets or custom color pickers.

## Use Cases

- Product color customization
- Design color selection
- Accent color choices
- Brand color matching


## Configuration

```
Preset Colors:
- Red (#FF0000)
- Blue (#0000FF)
- Green (#00FF00)
Allow Custom: Yes
```
