Date picker options provide a calendar interface for customers to select dates.

## Use Cases

- Preferred delivery date
- Event date for custom items
- Reservation dates
- Deadline selection


## Configuration

```
Min Date: Today + 3 days
Max Date: Today + 90 days
Blackout Dates: Holidays, weekends
```
