Image upload options let customers upload photos, logos, and design files for custom products.

## Use Cases

- Custom photo printing
- Logo for embroidery
- Design files for printing
- Reference images


## Configuration

```
Allowed Types: JPG, PNG, PDF
Max File Size: 10MB
Preview: Yes
```
