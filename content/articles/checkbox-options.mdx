Checkbox options let customers select multiple items from a list. Ideal for add-ons, extras, and multi-select preferences.

## Use Cases

- Product add-ons (batteries, cables, cases)
- Toppings and extras
- Service selections
- Feature preferences


## Configuration

Create multiple checkbox options with individual pricing:

```
☐ Extra Batteries (+$10)
☐ Carrying Case (+$15)
☐ Screen Protector (+$8)
```
