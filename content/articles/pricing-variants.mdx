Add extra charges to custom options and understand how Optionify works with Shopify's variant system.

## Adding Pricing to Options

Many option types support additional pricing:

```
Gift Wrapping: +$5.00
Express Shipping: +$15.00
Custom Engraving: +$10.00
```

## Price Configuration

For each option:

```
Enable Pricing: Yes
Price Amount: $5.00
Price Label: "Gift Wrapping Service"
Tax: Apply store tax settings
```


## Working with Shopify Variants

Optionify complements Shopify variants:

**Use Shopify Variants for:**
- Size, Color, Material (standard variants)
- Inventory tracking
- Different SKUs

**Use Optionify Options for:**
- Personalization (names, messages)
- Add-ons and services
- Customer preferences
- File uploads
