Textarea options provide multi-line text input fields perfect for collecting longer messages, detailed instructions, and customer notes that don't fit in a single-line text field.

## What Are Textarea Options?

Textarea options create a resizable, multi-line text box where customers can enter paragraphs of text. Ideal for:

- Long gift messages or cards
- Detailed special instructions
- Custom design descriptions
- Shipping notes and preferences
- Customer requirements or specifications

\![Textarea option on product page](/images/textarea-options/textarea-example.png)
*A textarea option for detailed gift messages*

## When to Use Textarea vs Text

| Use **Text** Option | Use **Textarea** Option |
|---------------------|------------------------|
| Names, initials | Paragraphs, multiple sentences |
| Short phrases (< 50 chars) | Long messages (> 50 chars) |
| Engraving text | Detailed instructions |
| Product codes | Design descriptions |
| Single-line input | Multi-line input |

## Creating a Textarea Option

### Step 1: Basic Configuration

```
Option Label: "Gift Card Message"
Internal Name: "gift-card-message"
Help Text: "Write a personalized message for your gift recipient"
Option Type: Textarea
```

### Step 2: Textarea-Specific Settings

#### Rows (Height)

Control the initial height:

```
Rows: 4 (Default - about 4 lines)
Rows: 6 (Taller - about 6 lines)
Rows: 10 (Very tall - for long content)
```

**Recommended heights:**
- Gift messages: 4-6 rows
- Instructions: 3-4 rows
- Design briefs: 8-10 rows

\![Different textarea heights](/images/textarea-options/textarea-heights.png)
*Choose the right height for your use case*

#### Character Limit

Always set a maximum:

```
Max Characters: 500 (Gift messages)
Max Characters: 1000 (Detailed notes)
Max Characters: 2000 (Design descriptions)
```


#### Character Counter

Show remaining characters:

```
Show Counter: Yes
Update: Real-time (as customer types)
Position: Below textarea
```

### Step 3: Validation & Requirements

```
Required: No (for optional messages)
Required: Yes (for critical specifications)

Minimum Characters: 10 (ensure meaningful input)
Maximum Characters: 500
```

### Step 4: Placeholder Examples

Good placeholders guide customers:

```
✅ "e.g., Dear Sarah, Happy 30th Birthday! We hope this
    year brings you joy and adventure. Love, Mom & Dad"

✅ "Describe your ideal design including colors, style,
    and any specific elements you want included..."

❌ "Enter your message"
❌ "Type here"
```

## Best Practices

### 1. Set Appropriate Limits

Match limits to use case:

```
Short notes: 200-300 characters
Gift messages: 400-600 characters
Instructions: 300-500 characters
Detailed specs: 1000-2000 characters
```

### 2. Provide Clear Guidance

```
Help Text: "Your message will be printed on a gift card
           included with your order. Maximum 500 characters."
```

### 3. Show Examples

Include a sample message:

```
Placeholder: "Example: Dear Mom, Thank you for everything
             you do. This gift is just a small token of
             my appreciation. Love always, Sarah"
```


### 4. Consider Formatting

Decide if you'll preserve:
- Line breaks
- Multiple spaces
- Special characters

```
Preserve Formatting: Yes (keeps line breaks)
Preserve Formatting: No (converts to single paragraph)
```

### 5. Mobile-Friendly Sizing

Test on mobile devices:

```
Mobile Rows: 4 (smaller on mobile)
Desktop Rows: 6 (larger on desktop)
```

## Advanced Features

### Auto-Resize

### Spell Check

Enable browser spell checking:


## Troubleshooting

### Text Gets Cut Off

**Problem:** Character limit too low
**Solution:** Increase max characters or show warning

### Textarea Too Small

**Problem:** Not enough rows
**Solution:** Increase row count or enable auto-resize

### Formatting Lost

**Problem:** Line breaks disappear
**Solution:** Enable "Preserve Formatting"

### Mobile Issues

**Problem:** Textarea unusable on phone
**Solution:** Set appropriate mobile row count

### No Character Counter

**Problem:** Counter not showing
**Solution:** Enable in settings, check theme compatibility
